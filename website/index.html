DrunkFTW

What is this:

This is a series of Terraria worlds that are both Drunk and For The Worthy.

Naming convention: DrunkFTW-XX
[X]X - World Size (L = Large; M = Medium; S = Small)
X[X] - Difficulty (J = Journey; C = Classic; E = Expert; M = Master)
